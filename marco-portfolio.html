Thesis Project: Upcoming

Description

This thesis focusses on the design and implementation of a vision-based object tracking system using a 2-DOF pan-tilt mechatronic platform. The system utilizes a camera connected to a processing unit to detect and track a moving object (ideally aerial object such as drone) in real time.

Development Stages

  • 1Implementing a basic tracking system capable of detecting and following a moving object.
  • 2Closed-loop control using tracking error, along with PID control to improve stability, accuracy, and response time.
  • 3Predictive algorithms such as the Kalman Filter to estimate the future position of the moving object.

Celestial Navigation System

Description

The Celestial Navigation System is a mechatronic platform designed to determine orientation by tracking celestial objects in real time. It uses a 2-DOF pan-tilt mechanism with a camera to observe the sky and identify sun or stars. By combining visual data with precise time information and orientation data from onboard sensors (IMU and magnetometer), the system estimates angular position and maintains accurate alignment.
